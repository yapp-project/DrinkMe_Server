# DrinkMe WEB Server

DrinkMe Web Server


## Skill Stacks

- MongoDB
- Mongoose
- Express generator
- dotenv
- body-parser


## Developer

- 김형준, 박지홍


## How to install

```$
$ git clone https://github.com/yapp-project/Drinkme_server.git
$ cd server
$ npm install
```
After you have to create '.env' and setting PORT, MONGo_URI (in app.js)


## How to run

```$
$ npm start 
```
Or, You can use node app.js
and enter to `localhost:YOUR_PORT`
